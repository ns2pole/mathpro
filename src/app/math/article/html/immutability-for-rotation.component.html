この記事で伝えたい主張：原点中心の回転変換によって実係数2次形式の判別式は不変である

必要な予備知識：数学3-２次曲線まで-

定義 2変数2次多項式


「2変数2次多項式$F(x,y)=ax^{{ '{' }}2{{ '}' }}+bxy+cy^{{ '{' }}2{{ '}' }}+dx+ey+f$の判別式」を$D(F(x,y))=b^{{ '{' }}2{{ '}' }}-4ac$ と表し、定義する。

命題　〜2変数2次多項式の判別式の回転不変性〜

任意の2変数2次多項式$F(x,y)=ax^{{ '{' }}2{{ '}' }}+bxy+cy^{{ '{' }}2{{ '}' }}+dx+ey+f$に対して、どんな$\theta \in \mathbb{{ '{' }}R{{ '}' }}$に対しても$D(F(xcosθ+ysinθ,-xsinθ+ycosθ))=b^2-4ac=D(F(x,y))$


Proof:
$F(xcosθ+ysinθ,-xsinθ+ycosθ)$
$=a(xcosθ+ysinθ)^{{ '{' }}2{{ '}' }}+b(xcosθ+ysinθ)(-xsinθ+ycosθ)+c(-xsinθ+ycosθ)^{{ '{' }}2{{ '}' }}+(1次以下の項)$
であるから、

$x^2の係数=a \cos ^2 \theta - b \cos \theta \sin \theta + c \sin ^2 \theta$

$xyの係数=2a \sin \theta \cos \theta + b \cos ^2 \theta - b \sin ^2 \theta - 2c \sin \theta \cos \theta$ $= a \sin 2\theta + b \cos 2 \theta - c \sin 2\theta = (a - c) \sin 2\theta + b \cos 2 \theta$

$y^2の係数=a \sin ^2 \theta + b \sin \theta \cos \theta + c \cos ^2 \theta$

よって、
$D(F(xcosθ+ysinθ,-xsinθ+ycosθ))$
$=((a - c) \sin 2\theta + b \cos 2 \theta)^2 - 4(a \cos ^2 \theta - b \cos \theta \sin \theta + c \sin ^2 \theta)(a \sin ^2 \theta + b \sin \theta \cos \theta + c \cos ^2 \theta)$
$=(\ $各自計算$)\ = b^2-4ac = D(F(x,y))$ $\ \ \ \ Q.E.D$
